To Run This Scenario, follow this command :
1. cucumber  : to only run the scenario
2. cucumber -f pretty --expand -f json -o report.json : run with generate report.json
3. Ruby Report_builder.rb : for generate report.html (after json file was created)

Terdapat 3 scenario :
1. Valid Data Login with Email Address
2. Valid Data Login with phone Number
3. Invalid Data login Using Worng Password 

 List Command: 
1. cucumber -t @ValidDataLogIn : Untuk menjalankan semua Scenario Valid data
2. cucumber -t @withEmailAddress : Untuk menjalankan scenario Valid Data with Email Address
3. cucumber -t @withPhoneNumber : Untuk menjalankan scenario Valid Data with Phone Number
4. cucumber -t @InValidDataLogIn / @UsingWrongPassword : Untuk menjalankan Scenarion Invalid Data
